# Free Codex Tutorial for Product Managers

![Codex for Product Managers](/images/codexforpms-thumbnail.png)

## Learn Codex IN Codex!

This course is taught entirely inside Codex itself 😈

- You'll install Codex, download the course files, then start the first lesson — and **Codex becomes your instructor**
- It guides you through each module interactively, right in your terminal
- You learn by doing real PM work with real files

## Who This Course Is For

**Product Managers** who want the most powerful way to put AI to work — not as a chat window you copy-paste into, but as an agent that actually does the work alongside you.

OpenAI Codex isn't just for engineers. For PMs it's an AI workspace that:
- Reads your research, drafts and edits PRDs, and analyzes data across many files
- Runs multi-step tasks and even works in parallel on batches of work
- Remembers context about your product through a project file (`AGENTS.md`)
- Acts as custom reviewers (engineer, exec, UX) before you share work with your team

**You don't need coding or terminal experience.** If you can describe what you want in plain English, you can use Codex.

## What You'll Learn

**Core Codex skills**
- Working with files and giving Codex the right context
- Running agents — and running several in parallel
- Building custom sub-agents that review your work from different angles
- Setting up `AGENTS.md` so Codex remembers your product and preferences

**Applied to real PM work**
- Write PRDs with multi-perspective feedback
- Analyze product data and read out experiments
- Develop strategy and build executive-ready slides
- Go from idea to a clickable, deployed prototype — no engineer required

## Course Structure

[**Module 0: Getting Started**](/getting-started/introduction) — Install Codex, sign in, and open the course. Your first run of an AI agent that actually executes PM tasks.

[**Module 1: Fundamentals**](/fundamentals/welcome) — Files, context, agents, sub-agents, and project memory — the core moves that make Codex useful, practiced on TaskFlow, your fictional company.

[**Module 2: Advanced PM Work**](/advanced/write-prd) — Write PRDs, analyze product data, and develop strategy using everything from Module 1.

[**Module 3: Vibe Coding**](/vibe-coding/setup) — Turn an idea into a working app you can click through, push it to GitHub, and deploy it live.

## What You'll Actually Do

**1.1 Welcome to TaskFlow** — Meet your fictional company and learn how the course works
**1.2 Visualizing Files** — Set up a workspace so you can see files change in real time
**1.3 First Tasks** — File references, research synthesis, images, and web search
**1.4 Agents** — Run independent Codex agents in parallel for batch work
**1.5 Custom Sub-Agents** — Build reusable specialists for multi-perspective review
**1.6 Project Memory** — Give Codex permanent context with `AGENTS.md`

**2.1 Write a PRD** — Partner with Codex to write better PRDs faster
**2.2 Analyze Data** — Funnel analysis, impact estimation, and A/B test readouts
**2.3 Product Strategy** — Make hard strategic calls and build executive slides

**3.1–3.5 Vibe Coding** — Plan, build, and iterate on a real app, save it to GitHub, and deploy it live with a public URL

## Prerequisites

- A ChatGPT account — a free tier exists; **Plus ($20/mo)** is recommended for comfortable daily use. [See Codex pricing](https://developers.openai.com/codex/pricing)
- Mac, Windows, or Linux computer
- Basic PM experience (you know what a PRD is)
- ~8 hours to complete the course
- Willingness to learn by doing

**No coding experience required.** If you can have a conversation and work with documents, you can use Codex.
